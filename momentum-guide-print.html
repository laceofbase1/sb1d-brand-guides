01 · The logo

The mark stays the anchor

The logo is the anchor of trust, so it never changes. Give it clear space, keep contrast, and on dark or busy fields place the full-color mark on a clean white field.

Primary icon
Horizontal lockup
On navy · white field
Clear spaceKeep space equal to the cap height on every side.
Minimum sizeIcon ≥ 28 px; horizontal logo ≥ 120 px wide on screen.
DoPlace on white, light neutrals, or a white field over color.
Don'tRecolor, redraw, stretch, or set straight onto a photo.

Azure passes AA for large UI and buttons with white text. For small body text on white, use Degree Navy or Warm Graphite, azure is a signal, not a paragraph color.

Pantone values are the closest spot-color matches, confirm against a physical Pantone guide before any print run. Use the coated (C) values for coated stock; ask for uncoated (U) equivalents if needed.

03 · Typography

Archivo, with Hanken Grotesk

Archivo, a heavy grotesque that matches the wordmark, carries every headline and label. Hanken Grotesk is its warmer counterpart for body and interface text.

Aa
Archivo · 700–900
Display, headlines, buttons, labels. Tight tracking, strong weight contrast.
Aa
Hanken Grotesk · 400–700
Body, captions, UI. Set 400–500 for paragraphs, 600–700 for emphasis.

04 · The Momentum system

Motion that means progress

Momentum is the brand's one rule made visible: every student is moving forward. Motion always travels ahead and always eases out, never bounces.

Progress, visible

Progress tracks fill navy-to-azure, left to right.

Forward cues

The azure arrow is the brand's gesture. It nudges ahead, never back.

See the path →

Rules of motion

One language everywhere.

Duration 180ms UI · 600ms reveals
Easing cubic-bezier(.16,1,.3,1)
Direction: left → right, up.

The whole system leans forward.

Arrows point ahead. Progress bars fill toward the goal. Reveals rise and settle. Nothing bounces, nothing drifts backward. If an element could suggest motion, it moves the same way the student does, forward.
